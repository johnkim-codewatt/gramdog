# 개선점
1. 유저가 영작 이외에 일반적인 문법 질문도 할수 있도록 한다.(학습관련 잘문 OK)
    예) 현재완료형에 대해 설명해줘.
2. 학습과 관련된 이야기가 아닐경우 "학습과 관련된 질문을 해주세요." 라고 응답한다.
    예) 오늘 날씨 어때? -> "학습과 관련된 질문을 해주세요."
3. 유저의 영작에 대한 오답 기록(DB)을 찾았다면 해당 오답 기록을 기반으로 피드백을 추가 제공한다. 
    예) 자주 틀리는 오답이네요. 지난번에도 관사 'the'를 빠뜨리셨는데, 이번에도 같은 실수가 보이네요. 주의해 보세요!
4. 정답/오답 에 대한 판단 기준을 조정(0~10까지)할수 있도록 한다. (변수로 지정하고 차후 개발자가 조정할수 있도록) -> 어느정도의 오답은 정답처리 할수 있도록
5. 문제 생성시 매번 똑같은 문제 X -> 다양하게 제공하도록 노력하고, 만약 유저가 "다른문제" 라고 입력하면 다른문제를 생성해 제공한다.
